GUI Overview

Qt GUI Client

Fleet Credits Qt GUI provides a full-featured graphical interface for the wallet and blockchain.

Files: src/qt/fleetcreditsgui.h, src/qt/fleetcredits.cpp, src/qt/fleetcreditsgui.cpp

Main Window

FleetCreditsGUI is the main application window:

  • Menu Bar: File, Settings, Help menus
  • Toolbar: Quick access to common functions
  • Status Bar: Network status, sync progress, connection count
  • Tab Interface: Overview, Send, Receive, Transactions, etc.

Core Models

ClientModel

Provides blockchain and network information to the GUI:

  • Block count and sync status
  • Network connections
  • Blockchain information
  • Update notifications

WalletModel

Manages wallet state and operations:

  • Balance and transaction history
  • Address management
  • Transaction sending
  • Wallet encryption status

TransactionTableModel

Provides transaction data for tables:

  • Transaction list with status
  • Sorting and filtering
  • Real-time updates

AddressTableModel

Manages address book:

  • Receiving addresses
  • Sending addresses
  • Address labels

Key Dialogs

SendCoinsDialog

Send Fleet Credits to addresses:

  • Multi-recipient support
  • Fee calculation and display
  • Coin control (UTXO selection)
  • Confirmation dialog

ReceiveCoinsDialog

Generate receiving addresses:

  • Address generation
  • QR code display
  • Payment request creation

OptionsDialog

Configure application settings:

  • Wallet settings
  • Network settings
  • Display preferences
  • Theme selection

CoinControlDialog

Select specific UTXOs for transactions:

  • UTXO list with details
  • Selection and filtering
  • Fee calculation preview

UI Components

FleetCreditsAmountField

Specialized input field for FC amounts:

  • Currency unit display (MFC, kFC, FC, mFC, ŒºFC)
  • Input validation
  • Format conversion

FleetCreditsAddressValidator

Validates Fleet Credits addresses:

  • Base58 validation
  • Network prefix checking
  • Checksum verification

Theme System

Fleet Credits GUI supports custom themes:

  • Theme Manager: Loads and applies themes
  • Theme Builder: Create custom themes
  • CSS Support: Stylesheet-based theming
  • Dark/Light Modes: Built-in theme variants

Features

  • Transaction History: Complete transaction list with filters
  • Address Book: Labeled addresses for easy access
  • RPC Console: Direct RPC command interface
  • Network Monitor: Connection status and peer information
  • Mining Status: Block generation status
  • Contribution Interface: Submit and view contributions
  • MWEB Interface: Peg-in/peg-out operations

Currency Units

Fleet Credits GUI displays amounts in multiple units:

  • MFC: Mega Fleet Credits (1,000,000 FC)
  • kFC: Kilo Fleet Credits (1,000 FC)
  • FC: Fleet Credits (base unit)
  • mFC: Milli Fleet Credits (0.001 FC)
  • ŒºFC: Micro Fleet Credits (0.000001 FC)
