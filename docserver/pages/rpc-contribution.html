Contribution RPC

RPC commands for submitting and querying contribution transactions

Overview

The Contribution RPC API allows you to submit contribution transactions to the Fleet Credits network and query their status. Contribution transactions can represent code contributions, charitable acts, creative work, educational content, mentorship, AI validation, data labeling, or ethical reviews.

Endpoints

POST submitcontribution
Submit a contribution transaction to the Fleet Credits network.

Parameters

NameTypeRequiredDescription
contrib_typestringYesContribution type: CODE_CONTRIBUTION, CHARITABLE_ACT, CREATIVE_WORK, EDUCATIONAL_CONTENT, CODE_MENTORSHIP, AI_VALIDATION, DATA_LABELING, ETHICAL_REVIEW
proof_datastringYesProof data (GitHub commit hash, IPFS hash, etc.)
metadatastringNoAdditional metadata JSON string

Response

{
  "txid": "hash",
  "status": "pending",
  "message": "Contribution transaction created"
}

Example

fleetcredits-cli submitcontribution "CODE_CONTRIBUTION" "abc123def456..."
fleetcredits-cli submitcontribution "AI_VALIDATION" "ipfs_hash_here" '{"accuracy_score": 0.95}'
GET getcontributionstatus
Get the status of a contribution transaction.

Parameters

NameTypeRequiredDescription
txidstringYesThe contribution transaction ID

Response

{
  "txid": "hash",
  "status": "verified",
  "bonus_level": "HIGH",
  "reward": 11500
}

Example

fleetcredits-cli getcontributionstatus "abc123..."
GET listcontributions
List contribution transactions, optionally filtered by address or type.

Parameters

NameTypeRequiredDescription
addressstringNoFilter by contributor address
contrib_typestringNoFilter by contribution type

Response

[
  {
    "txid": "hash",
    "type": "CODE_CONTRIBUTION",
    "status": "verified",
    "reward": 1000
  },
  ...
]

Example

fleetcredits-cli listcontributions
fleetcredits-cli listcontributions "FleetCreditsAddress" "AI_VALIDATION"

Contribution Types

TypeValueDescription
CODE_CONTRIBUTION0x01Open-source code commits
CHARITABLE_ACT0x02Verified charitable donations
CREATIVE_WORK0x03Art, writing, music with IPFS hash
EDUCATIONAL_CONTENT0x04Teaching materials or courses
CODE_MENTORSHIP0x05Mentor-mentee code reviews
AI_VALIDATION0x06Human validation of AI outputs
DATA_LABELING0x07Dataset labeling with inter-annotator agreement
ETHICAL_REVIEW0x08Ethical review (requires MWEB)

Implementation Status

Note: The contribution RPC commands are currently implemented with placeholder functionality. Full implementation includes:
  • Transaction signing with wallet keys
  • Proof data hash calculation
  • Mempool submission
  • Network broadcast
  • Blockchain query for status
  • Verification record lookup
