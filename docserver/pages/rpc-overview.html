Authentication

RPC calls require authentication via rpcuser and rpcpassword set in fleetcredits.conf:

rpcuser=your_username
rpcpassword=your_password
rpcport=42068
rpcallowip=127.0.0.1

Example Usage

Using fleetcredits-cli

fleetcredits-cli getblockcount
fleetcredits-cli getnewaddress
fleetcredits-cli submitcontribution "CODE_CONTRIBUTION" "abc123def456..."

Using curl

curl --user rpcuser:rpcpassword \
  --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getblockcount", "params": []}' \
  -H 'content-type: text/plain;' \
  http://127.0.0.1:42068/
